20190307_TIL

Posted on 2019-03-07 | Post modified: 2019-03-07 | 0 comments
Words count in article: 600 | Reading time ≈ 1 mins.

일주일동안, 아니지 원랜 내일까지, multiprocessing, openMP, MPI 이것들을 C와 Fortran으로 공부했다. 주로 fortran으로 학습하고 c로는 그냥 코드 이해하고 컴파일해보고 비교분석만 했고 주 코더가 될 Python으로는 openMP, MPI가 어떤 식으로 있고 사용되고 있는지 찾아봐야겠다. 벌써 3달쨰인가? 서울서 대전까지 스스로 내 차비내고 숙박비 내면서 격주 마다 일주일동안 먹고 자고 생활하면서 교육받은 보람이 참 있는 것 같다. 항상 혼자 해결하고 삽질했었는데 이렇게 현업에서 국내에서 가장 빠른 컴퓨터를 직접 구동하면서 실습하고 전문가가 옆에서 바로 바로 잡아주니 정말.. 너무 좋았다. 이제 이걸 내껄로 만드는 것만 남았고 또 응용해서 내가 추구하고자 하는 방향에 어떻게 접목시킬까는 나중문제지만 어쨌든… 앞으로 일주일짜리 2번 하고도 하루 더 남은 교육, 잘 받고 잘 정리해서 머리속에 차곡차곡 쌓아놓쟈.

그동안 TIL 은 github private에 쌓았지만… 조금 더 정리하고 스켄도 뜨고 이해가 되면 언제고 public으로 돌려야지.

하고 싶은 것도 많고 할 것도 많고 공부할 것도 많고 아쉬운 시간은 흘러만 가고..
