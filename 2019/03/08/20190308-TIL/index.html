fortran 그리고 pthread

PuTTY에서 리눅스 서버 접속 후 사용하다가 내 컴퓨터 windows 환경에 직접 모듈, 컴파일러, 그리고 에디터를 연결해서 사용하려 했다. 에디터는 vs code 로 기존에 쓰고 있는 것을 사용할 것이었기에 컴파일러만 opensorce로 되어있는 gcc, gfortran, 그리고 상업컴파일러 intel 을 설치하려했다. (linux나 mac 도 아래 링크 다운 받는 곳에 각 플렛폼 따라 받으면 됨)

컴파일러 설치

GNU

GCC.GNU.ORG 여기서 최신 버전을 확인하고 오늘자 2019년 3월 8일 최신 released인 GCC 8.3을 받았다.

  • 버전 https://gcc.gnu.org/releases.html
  • 다운 https://gcc.gnu.org/mirrors.html

이걸 이제 어떻게.. 사용하는지.. 전혀 몰랐기에.. 이리 저리 뒤적거리며 시도해보다 https://gcc.gnu.org/wiki/GFortran 이 곳에서 fortran, gfortran, compiler에 OpenMP까지 다양한 정보가 있는 것을 확인! 찾아들어갔다.

  • 버전 https://gcc.gnu.org/wiki/GFortran/News
  • 다운 https://gcc.gnu.org/wiki/GFortranBinaries#Windows

위 사이트 Windows에 보면

  1. MinGW for Win64
  2. MinGW (“native Windows” build)
  3. TDM GCC: MinGW/MinGW-W64 Builds
  4. Cygwin: The Cygwin project offers up-to-date builds of GCC and gfortran

이렇게 4가지가 있는데 MinGW for Win64 에 Mingw-w64로 64비트에 각종 칩설정 부터 이것저것 시도하다 포기했고 MinGW로 다시 다운 받아서 설치했다.

http://www.mingw.org/ 이 홈페이지 download/installer 창 주소가 아래

  • 주소 https://osdn.net/projects/mingw/downloads/68260/mingw-get-setup.exe/

실행해보니 참 종류가 많았다. 처음 몇개만 설치해서 시도하다가 안되서 basic setup 몽땅 설치해서 시도했따.
